# Delivery ETA & Demand Forecasting

End-to-end SQL, Python, and machine learning project analyzing order flow data to predict delivery times and forecast demand hotspots.

## Tech Stack
- MySQL
- Python
- scikit-learn
- pandas
- Jupyter

## Project Structure
- data/ – datasets
- sql/ – analysis queries
- notebooks – EDA & modeling
- src – scripts
- reports – insights & plots
